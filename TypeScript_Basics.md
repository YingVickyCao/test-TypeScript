## Accessibility modifier

| accessibility modifier | Acees Level                                        |
| ---------------------- | -------------------------------------------------- |
| private                | inside of class                                    |
| protected              | inside of class and subclass                       |
| public                 | Anywhere: inside and outside of class and subclass |
